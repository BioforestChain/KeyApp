内存泄漏 ​

  • MUST 页面卸载后释放所有订阅
  • MUST 组件卸载后清理定时器
  • MUST 大对象使用后置 null
  • SHOULD 定期检查内存增长

动画性能 ​

动画帧预算 ​

时间用途
16ms一帧（60fps）
< 10msJS 执行
< 4ms渲染+合成
< 2ms余量

动画规范 ​

动画类型实现方式说明
页面切换CSS transformGPU 加速
列表滚动原生滚动避免 JS 干预
按钮反馈CSS transition< 100ms
加载动画CSS animation避免 JS

禁止的动画模式 ​

  • MUST NOT 动画中修改布局属性（width/height）
  • MUST NOT 动画中触发重排（offsetTop 等）
  • MUST NOT 使用 setInterval 做动画
  • SHOULD NOT 同时运行超过 3 个动画
