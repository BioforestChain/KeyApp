## 构建命令

```bash
# SDK
pnpm sdk:build           # 构建 SDK
pnpm sdk:dev             # 监听模式

# 小程序
pnpm miniapps:build      # 构建所有小程序
pnpm miniapps:dev:teleport   # 开发一键传送
pnpm miniapps:dev:forge      # 开发锻造

# 测试
pnpm test -- --testPathPattern=ecosystem
```
