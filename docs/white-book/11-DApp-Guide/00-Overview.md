# 00. 生态架构 (Overview)

KeyApp 旨在构建一个去中心化的 DApp 操作系统。我们不只是提供一个浏览器，而是提供一个**运行时 (Runtime)**。

## 接入方式

1.  **Miniapp (推荐)**: 遵循 KeyApp Manifest 规范的 Web 应用。可以获得完整的 OS 能力（如多窗口、深色模式同步、原生通知）。
2.  **Standard Web App**: 普通的 H5 应用。在受限的浏览器容器中运行，可以使用标准的 Wallet 连接协议。

## 核心价值

*   **Discoverability**: 通过去中心化的 App Store 分发。
*   **Native Experience**: 像原生应用一样运行，而不是像网页一样运行。
*   **Security**: 基于 Manifest 的权限声明系统，用户明确知道 DApp 请求了什么。
