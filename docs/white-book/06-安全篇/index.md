# 第六篇：安全篇

> 定义产品的「如何保护用户」

---

## 概述

安全篇回答三个核心问题：

1. **密钥如何保护？** → [密钥管理](./01-密钥管理/)
2. **身份如何验证？** → [身份认证](./02-身份认证/)
3. **DApp 如何授权？** → [DWEB 授权](./03-DWEB授权/)

---

## 章节导航

### [第十八章：密钥管理](./01-密钥管理/)

定义助记词、私钥的生成和存储方案。

**要点**：
- BIP39 助记词生成
- 多链地址派生 (BIP44/Ed25519)
- 加密存储方案

### [第十九章：身份认证](./02-身份认证/)

定义用户身份验证机制。

**要点**：
- 应用锁（密码/指纹）
- 指纹支付
- 自动锁定策略

### [第二十章：DWEB 授权](./03-DWEB授权/)

定义 Plaoc 协议的授权流程。

**要点**：
- 地址授权
- 签名授权
- 授权管理
